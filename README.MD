# 💈 Sistema de Agendamento para Barbearia

Projeto desenvolvido em Python com integração futura com WhatsApp.

## 🚀 Funcionalidades

- Agendamento de horários
- Bloqueio de horários duplicados
- Regras de funcionamento (domingo e segunda)
- Listagem de horários disponíveis

## 🛠️ Tecnologias

- Python
- SQLite
- Flask
- Twilio API

## 📲 Em desenvolvimento

- Integração com Google Agenda
