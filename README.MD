# 💈 Sistema de Agendamento para Barbearia

Desenvolvi um sistema de agendamento para barbearia utilizando Python, com foco na automação do processo de marcação de horários.

O sistema permite que clientes escolham datas e horários disponíveis, evitando conflitos através de validação automática no banco de dados. Também inclui regras reais de funcionamento, como limitação de horários aos domingos e fechamento às segundas-feiras.

O projeto foi estruturado de forma modular, separando lógica de negócio, manipulação de dados e execução principal, seguindo boas práticas de desenvolvimento.

Futuramente, o sistema poderá ser integrado com WhatsApp e Google Agenda para automação completa do atendimento.

## 🚀 Funcionalidades

- Agendamento de horários
- Bloqueio de horários duplicados
- Regras de funcionamento (domingo e segunda)
- Listagem de horários disponíveis

## 🛠️ Tecnologias

- Python
- SQLite
- Flask
- Twilio API

- Integração com Google Agenda
